# MCAFK
> 轻量但高级的挂机的插件
<img src="intro.png">

> 其实玩家头顶也会显示 [挂机中]。

# 插件使用方法
`/afk` 挂机/取消挂机
> 挂机过程中不可被攻击/移动
`/afk {玩家}` 管理员开关玩家挂机状态

# 支持版本
> 1.21+（已测试）<br/>其他版本如无法使用请安装maven修改配置编译，您可以直接使用export.bat进行编译。
